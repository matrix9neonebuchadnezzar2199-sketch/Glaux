# Glaux Context & History Policy

## 履歴

- チャットは**記録しない**
- 履歴ファイル、DB、復元機能なし
- 会話はプロセス内メモリのみ
- 破棄タイミング: アプリ終了、クリーン、（任意）モデル再起動

## コンテキスト概算

```
estimated_tokens = ceil(total_chars / 2)
```

system + user + assistant メッセージの文字列合計から算出。

| しきい値 | 動作 |
|----------|------|
| 70% | caution 表示 |
| 90% | Alert「クリーン推奨」 |
| 100% | 送信前確認（送信 or クリーン） |

v1 では自動要約・自動圧縮なし。

## クリーン

- メモリ上の messages 配列を空にする
- UI の Message list を初期化
- 推論中は先に停止してからクリーン

## Help 記載事項

- チャットは保存しない
- 履歴はない
- コンテキスト表示は概算
- 長文会話は想定外 → クリーンを使う
